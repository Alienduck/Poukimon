# Project Poukimon

_Poukimon is a short project to learn basics of React_

## Setup

1. Clone the project repository:
   ```bash
   git clone https://github.com/your-username/poukimon.git
   ```
2. Navigate to the project directory:
   ```bash
   cd poukimon
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
   _You can use pnpm if you need_

## Getting Started

To run the project in development mode:

```bash
pnpm run dev
```

## Usage

On the website you'll found:

- Home page (landing)
- Poukidex (list of all poukimons)
- Randomizer (generate random poukimon)
- Game (guess the poukimon by it's shadow)
- My Team (your favorites poukimons)
